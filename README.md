# ✝ 일신봇

일신교회 청년부 디스코드 봇

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/입장` | 음성 채널 입장 |
| `/퇴장` | 음성 채널 퇴장 |
| `/재생` | 음악 재생 |
| `/일시정지` | 일시정지 |
| `/계속` | 다시 재생 |
| `/정지` | 정지 |
| `/스킵` | 다음 곡 |
| `/재생목록` | 목록 보기 |
| `/비우기` | 목록 비우기 |
| `/셔플` | 섞기 |
| `/반복` | 현재 곡 반복 |
| `/전체반복` | 목록 반복 |
| `/현재곡` | 현재 곡 |
| `/볼륨` | 볼륨 조절 |
| `/도움말` | 도움말 |

## PM2

```
pm2 start ecosystem.config.js
pm2 restart 일신봇
pm2 stop 일신봇
pm2 logs 일신봇
```
